# Portfolio Personal - Práctica de HTML

Este repositorio contiene la entrega del Trabajo Práctico correspondiente a la Semana 4 del trayecto formativo en el Hospital Universitario. El proyecto consiste en la creación de una página web estática utilizando HTML5 para estructurar un portfolio personal.

## 📌 Objetivos del Proyecto

*   Aplicar la estructura semántica de un documento HTML (`header`, `section`, `footer`).
*   Implementar correctamente etiquetas de texto, listas e imágenes.
*   Desarrollar un formulario de contacto funcional con validación nativa.
*   Establecer las bases estructurales para la futura integración de hojas de estilo (CSS) y diseño responsive.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura y semántica de la página.
*   **Git / GitHub:** Control de versiones y alojamiento del repositorio.

## 🚀 Cómo visualizar el proyecto

Al ser un proyecto estático básico, no requiere la instalación de dependencias ni servidores locales. 

1. Clona este repositorio en tu equipo local:

   git clone [https://github.com/Scomes02/Tp_HTML.git](https://github.com/Scomes02/Tp_HTML.git)

2. Navega hasta el directorio del proyecto.

3. Haz doble clic en el archivo index.html para abrirlo en tu navegador web predeterminado.
